Engineering Decisions

Four deliberate architectural choices

1

One agent = one responsibility

Each agent is independently testable and reusable. Swap one without touching the rest.

2

Validation without LLMs

Platform rules are deterministic. A character count is either over the limit or it isn't — no model needed, no hallucination risk.

3

Three structured output strategies

JSON mode for the Planner · Custom delimiters for captions · XML tags for the Chat Agent — matched to each agent's output shape.

4

Two-step image generation

GPT-4o writes a context-rich prompt first, then DALL-E 3 renders it. Better images than a raw user description every time.
